# ✍️ 프로젝트 개요

전공교재의 가격이 부담되는 대학생들을 위해 중고책 거래와 그 교재와 연계된 스터디 모집 서비스를 제공하는 시스템이 필요하다고 생각하였습니다. <br>

#### 📝 중고거래 📝 공동구매 📝 스터디모집 📝 중고책 경매 📝 구매자와판매자 채팅
