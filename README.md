
# ✍️ 프로젝트 개요

‘북치고 중고치고’ 는 
대학생들이 사용하던 중고책을 판매할 수 있는 거래 플랫폼입니다. <br>
 1) ```경매 기능```  입찰자는 입찰을 신청할 수 있고, 판매자는 경매를 마감할 수 있습니다.<br>
 2) ```스터디 그룹 기능``` 스터디 호스트는 특정 책을 가지고 스터디원을 모집할 수 있습니다. <br>
 3) ```채팅 기능``` 판매자와 구매자간 채팅 , 스터디원 끼리 단체 채팅 기능을 지원합니다.<br>
 4) ```도서 공동구매기능``` 사용자는 원하는 도서를 다른 사용자와 공동구매를 할 수 있도록 지원합니다.<br>
이외에도 ```로그인 및 회원가입 기능```, ```개인 정보 관리 및 탈퇴 기능```, ```도서 즐겨찾기 기능```과 같은 기능이 존재합니다.<br><br>
 
 ### ```북치고 중고치고``` 는 ```Spring MVC```을 활용하여 개발하였습니다.<br>
 궁극적으로 이 웹 페이지는 사용자들끼리 중고책을 거래할 수 있는 더 나은 환경을 제공하기 위하여 개발되었습니다.<br>
